# Eglah-Must-Fintech

## Description

This system provides functionalities for managing various aspects of P2P investments and loans, including information management, investment type management, deposit/withdrawal details, sales history inquiry, investment details inquiry, bond details viewing, SMS management, consultation history management, and 1:1 inquiry history inquiry.

## Features

Information Management
Investment Type Management
Deposit/Withdrawal Details Checking
Sales History Inquiry
Investment Details Inquiry
Bond Details Viewing
SMS Management
Consultation History Management
1:1 Inquiry History Inquiry

## Usage

To use this system, simply open the index.html file in a web browser. Navigate through the different sections using the sidebar links.
